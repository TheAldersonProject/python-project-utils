Python Project Utils

Python Project Utils is a comprehensive toolkit designed to streamline Python project management and development workflows. This library provides utilities for handling configuration files, managing project structures, and automating common development tasks.

Overview

The Python Project Utils library offers a collection of tools that help developers:

  • Parse and manipulate configuration files (TOML, YAML)
  • Manage project dependencies and requirements
  • Automate project setup and maintenance tasks
  • Standardize project structures and workflows

Getting Started

To use Python Project Utils in your project:

from project_tools.core import file_handler

# Parse a TOML configuration file
config = file_handler.get_toml_file_content("pyproject.toml")

# Parse a YAML configuration file
yaml_config = file_handler.get_yaml_file_content("config.yaml")
